# AhgFilmes

Projeto para buscar filmes

## Descrição

Esse projeto busca os filmes que o usuario procurar. 

## Iniciando

É necessário ter algum gerenciador de arquivos
ex:npm ou yarn.

### Instalando as dependencias
```
npm i 
```
### executando o programa
```
npm start 
```
