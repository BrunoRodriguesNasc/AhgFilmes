# AhgFilmes

Projeto para buscar filmes

## Description

Esse projeto busca os filmes que o usuario procurar. 

## Getting Started

É necessário ter algum gerenciador de arquivos
ex:npm ou yarn.

### Installing
```
npm i 
```
para instalar as dependencias 

### Executing program

```
npm start 
```
para rodar o programa
